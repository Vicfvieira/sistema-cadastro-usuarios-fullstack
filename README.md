# Sistema de Cadastro de Usuários – Full Stack

## Descrição
Projeto Full Stack desenvolvido com o objetivo de consolidar os fundamentos de desenvolvimento web, aplicando conceitos de integração entre front-end e back-end, APIs REST e banco de dados relacional.

## Objetivo
Praticar a criação de um sistema CRUD (Create, Read, Update, Delete), simulando um cenário real de aplicação corporativa.

## Tecnologias Utilizadas
- Back-end: C# | .NET
- Front-end: React | TypeScript | HTML | CSS
- Banco de Dados: SQL Server
- Versionamento: Git | GitHub

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Edição de dados
- Exclusão de registros

## Aprendizados
- Criação de APIs REST
- Integração entre front-end e back-end
- Persistência de dados
- Boas práticas de organização de código

## Status
Em desenvolvimento para fins de aprendizado.
